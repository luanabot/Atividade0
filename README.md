Programa: O Localizador de Extremos (Com Arrays)

O objetivo era testar iteração simples e condicionais, escrevendo um método em Java que receba um array de inteiros 'int[] vetor' e imprima o maior valor, o menor valor e a média aritmética dos elementos.

Observação: O tamanho do vetor deve ser definido pela pessoa utilizadora.

Preencha o vetor com números gerados aleatóriamente, em formato inteiro.

Não é permitido o uso de bibliotecas para agilizar o processo.
